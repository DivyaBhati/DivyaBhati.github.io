ATS (Automated Trading Systems) October 2017 - Present
ATS is a project that myself and an assigned group chose to work on for a course called CS 196. The goal of this project is to develop a website in which a user can enter two stock symbols and specify a time period to analyze them over. Then the user will get presented with a graph displaying the stocks over the specified period along with an analysis of any possible correlation between the stocks. For now we don't have a registered domain for the site but you can download it from the repository and run it from your computer!
